# HumiPure - Industrial Humidification Systems

A modern marketing website for HumiPure's industrial-grade hybrid adiabatic humidification systems.

## Features

- Interactive psychrometric (Mollier) diagram visualization
- Responsive design with smooth animations
- Technical specifications and documentation
- Contact form for custom quotes

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build

```bash
npm run build
```
